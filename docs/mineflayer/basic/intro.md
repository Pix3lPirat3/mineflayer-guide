# Introduction

Once you've got your project loaded you can add the [Mineflayer package](https://www.npmjs.com/package/mineflayer) by using
```bash copy
npm i mineflayer
```

## Vanilla Require

You can then load the package into your project using `require()`
```js copy
let mineflayer = require('mineflayer');
````
